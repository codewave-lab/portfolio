무엇을 만들었는지 -
pt헬스장 랜딩페이지

어떤 역할 했는지 -
의뢰인의 시안을 받아 요구사항을 반영하여 랜딩페이지 제작

어떤 기술 썼는지 -
SEO 및 오픈 그래프(OG) 최적화
디자인 시스템 및 CSS 토큰화 구축
Design Token 기반의 반응형 UI 시스템 구축
웹 접근성 (A11y) 고려: 아이콘으로만 이루어진 버튼(예: 햄버거 메뉴)에 aria-label 속성을 부여하여 
시각 장애인용 스크린 리더기의 접근성을 높임.
CSS Custom Properties와 4/8pt Grid System 기반의 디자인 시스템 구축
자바스크립트를 활용해 가벼운 DOM 제어 및 인터랙티브 UI(모바일 메뉴 토글 등) 구현
모던 UI/UX 레이아웃 설계: * position: sticky와 scroll-padding-top을 조합하여
콘텐츠를 가리지 않는 쾌적한 상단 고정 내비게이션(GNB) 및 스무스 스크롤링 구현.
유지보수성 향상 및 개선 


어떤 문제 해결했는지 -
반응형 구현시 각 브레이크 포인트에서 홀수 카드의 배열을 고민,
와이어프레임이 손상되는 구간에 각각 브레이크 포인트를 잡아
3열에서 2열 1열로 순차 정렬을 구현하여
와이어프레임이 망가지는 경우를 방지함.
