[프로젝트 에필로그] La Vien 프리미엄 피부과 랜딩페이지

무엇을 만들었는지
피부 본연의 아름다움을 디자인하는 프리미엄 피부과 'La Vien(라비앙)'의 브랜드 가치를 전달하는 반응형 랜딩페이지를 제작했습니다. 고급스럽고 신뢰감 있는 UI/UX를 목표로, 시각적인 피로도를 낮추는 부드러운 인터랙션과 직관적인 정보 전달에 중점을 두어 개발했습니다.

어떤 역할을 했는지
기획부터 퍼블리싱, 인터랙션 구현까지 프론트엔드 전 과정을 담당했습니다. AI(Gemini)와의 협업 및 페어 프로그래밍을 통해 실제 상업 사이트의 레퍼런스를 분석하고, 이를 바탕으로 웹 표준과 접근성을 준수한 시맨틱 마크업 및 최적화된 스크립트 로직을 구현했습니다


어떤 기술을 썼는지
-Core: HTML5, CSS3, ES6 JavaScript

-Libraries: * Swiper.js (모바일 터치 및 스와이프에 최적화된 무한 슬라이더 구현), Intersection Observer (성능 최적화된 스크롤 리빌 애니메이션)

-Lenis.js (브라우저 간 편차를 줄인 우아한 Smooth Scroll 적용)

-Architecture: CSS Variables(디자인 토큰) 기반의 유지보수가 용이한 스타일 구조화, 4단계 콘텐츠 중심 브레이크 포인트(1310px, 1075px, 768px, 500px)를 활용한 완벽한 반응형 웹(RWD) 구축

어떤 문제 해결했는지

반응형 리테이닝 (Retaining) 해결: 
기기별 해상도 대응 시 발생하는 레이아웃 붕괴를 막기 위해 CSS clamp와 float, absolute 구조를 정석적인 flex/grid와 swiper로 마이그레이션하여 모바일 터치 환경에서 발생하던 가로 스크롤 이슈(overflow-x)를 해결했습니다.  

슬라이더 인터랙션 및 반응형 충돌 디버깅: 초기 하드코딩된 JS 애니메이션이 브라우저 리사이징 시 CSS의 크기 변화를 따라가지 못해 슬라이드가 어긋나는 이슈가 있었습니다. 이를 해결하기 위해 반응형 연산과 모바일 터치 제어에 특화된 Swiper.js로 마이그레이션하여, PC의 마우스 드래그와 모바일 스와이프를 모두 지원하는 안정적인 슬라이더 경험을 구축했습니다.

UX 최적화: PC와 모바일의 스크롤 속도 체감 차이를 분석하여, 모바일 환경에 맞는 '스크롤 리빌' 애니메이션 속도와 딜레이를 최적화했습니다. 사용자가 스크롤을 내릴 때 화면이 비어있지 않도록 시각적 정보가 즉각적으로 등장하는 타이밍을 미세 조정하여 몰입도를 극대화했습니다.  

사용자 친화적 인터페이스: 로그인 모달의 외부 클릭 닫기, 모바일 최적화된 언어 선택 드롭다운 등을 통해 직관적인 팝업 레이어 시스템을 구축하여 복잡한 페이지 이동 없이 사용자가 핵심 액션(로그인, 상담 신청)에 바로 접근할 수 있도록 설계했습니다.  

스크롤 애니메이션과 브라우저 기본 동작의 조화: 히어로 섹션의 화면 축소(Shrink) 효과와 부드러운 스크롤을 동시에 적용하는 과정에서 휠 이벤트 가로채기(Hijacking)로 인한 화면 멈춤 현상이 발생했습니다. 이를 JS 휠 이벤트 강제 제어에서 Lenis 라이브러리의 native scroll 연동 방식으로 구조를 변경하여, 스크롤바와 화면 이동 간의 딜레이를 없애고 고급스러운 스크롤 감성을 완성했습니다. 동시에 모바일 기기에서의 object-fit: cover 이미지 초점 어긋남 문제를 CSS object-position 연산으로 보정하여 모바일 사용성까지 디테일하게 챙겼습니다.
